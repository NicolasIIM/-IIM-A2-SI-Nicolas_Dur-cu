Transition

Créer un effet de transition entre plusieurs états d'un même élément.

.element {
transition-property : [propriété];
transition-duration : [durée];
transition-timing-function : [effet];
transition-delay : [retard];
}

Keyframes & Animations

Créer une réelle animation sur un élément uniquement en CSS.
Ces 2 propriétés CSS ne peuvent pas s'utiliser l'une sans l'autre

@keyframes monAnimation {
0%, 100%{
background-color:
white;
}
50%{
background-color:
pink;
}
}

.element{

animation-name:[nomDuKeyframe];
animation-duration:[durée];
animation-timing-function:[effet];
animation-delay:[retard];
animation-iteration-count:[iteration];
animation-direction:[direction];
animation-fill-mode:[spécificitéFin];
animation-play-state:[état];

}

SVG

Un fichier SVG est un fichier image dont les éléments qui le composent peuvent s'animer.

Un fichier SVG n'est pas dégradé même quand il est zoomé.
